개요

  1. 외부 sdk 없이 pionnet 서버로 push 발송하는 서비스
  2. fcm 토큰을 받아서 pionnet api에 토큰 등록
    http://10.74.105.65:22121/token/regist?t=<토큰값>
  4. 개발자 도구의 Network 탭을 열고 ESC 키를 눌러서 콘솔창도 열어주세요.
  5. 실행버튼을 누르면 자바스크립트 코드가 실행됩니다.

생성 - POST

콜렉션 읽기 - GET

부분 읽기 - GET

부분수정 - PATCH

전송하지 않은 데이터는 삭제되지 않는 것에 주목해주세요.

전체수정 - PUT

전송하지 않은 데이터는 삭제되는 것에 주목해주세요.

삭제 - DELETE
